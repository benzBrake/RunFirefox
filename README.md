# RunFirefox

从 MyFirefox 修改而来，是 Firefox 的便携版引导器

1. 自定义Firefox浏览器程序文件、数据文件夹、缓存文件夹的位置等。
2. 制作Firefox便携版，可设为默认浏览器（与安装版一样，在浏览器设置里设置即可）。
3. 支持浏览器启动/退出时运行外部程序。
4. 支持锁定到任务栏后点击启动（打开浏览器后在任务栏右键锁定即可）
